:: SpringBoot :: Swagger ::
===========================

# Overview

This challenge is open to learn and experiment with SpringBoot and Swagger to document RESTful APIs.

- [Rest API tooling review](https://opencredo.com/rest-api-tooling-review/)

# Resources

- [Swagger](http://swagger.io/) is a powerful open source framework backed by a large ecosystem of tools that helps you design, build, document, and consume your RESTful APIs.
    - [swagger-api/swagger-core](https://github.com/swagger-api/swagger-core) is the Java implementation of Swagger. Current version supports JAX-RS and plain servlets.
- [Swagger UI](http://swagger.io/swagger-ui/) allows visualizing and interacting with the API's resources without having any of the implementation logic in place.
    - [swagger-api/swagger-ui](https://github.com/swagger-api/swagger-ui) is a dependency-free collection of HTML, Javascript, and CSS assets that dynamically generate beautiful documentation from a Swagger-compliant API.
