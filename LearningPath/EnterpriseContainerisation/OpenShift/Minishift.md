:: OpenShift :: Minishift ::
============================

# Overview

## What is Minishift?

> Minishift is a tool that helps you run OpenShift locally by running a single-node OpenShift cluster inside a VM. You can try out OpenShift or develop with it, day-to-day, on your local host.
